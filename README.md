AlumniConnect — Simple Alumni–Student Interaction Platform
    AlumniConnect is a lightweight and responsive web platform that helps students connect with alumni for mentorship, career guidance, and      networking. It includes a secure login system, a searchable alumni directory, and a modern UI built completely using HTML, CSS, and          JavaScript.
Key Features
    Student login with roll number validation
    Page protection using browser localStorage
    Logout functionality
    Live search filter for alumni (name, company, skills)
    Responsive and clean user interface
    Fully implemented using HTML, CSS, and JavaScript
